# PIPT Repertório

Este é um projeto **pessoal** — repertório musical do Ministério de Música da PIPT (Primeira Igreja Presbiteriana de Taguatinga). Não tem nenhuma relação com a Wealthsimple.

## Regra rígida: nada de Wealthsimple neste projeto

Não use, mencione, ou tente conectar a qualquer ferramenta, sistema ou contexto da Wealthsimple. Isso inclui:

- **MCPs proibidos aqui**: `mcplocker` (e todos os sub-servidores `mcplocker__*`), `atlassian`, `glean`, `datadog`, `sentry`, `backstage`, `slackv2`, `jira`, `notion` (workspace WS).
- **Não** procure tickets JIRA (WPE, INGEST etc.), páginas Confluence, canais Slack, dashboards Datadog, deploys Backstage.
- **Não** aplique convenções, padrões de PR, regras de code review, ou fluxos internos da Wealthsimple (ex: `ws-compass`, seed-PR flow, feature flags, stacked PRs corporativos).
- **Não** invoque skills ou plugins do `@ws-claude-marketplace`.
- **Não** faça referência ao monorepo `front-end-monorepo`, código proprietário, ou memórias/hábitos capturados no contexto de trabalho.

Se precisar de contexto externo, use apenas ferramentas públicas: web search, fetch, GitHub público, documentação pública.

## O que é este projeto

Site estático + dados do repertório musical da igreja. Stack simples, HTML/CSS/JS. Veja `README.md` para estrutura.
